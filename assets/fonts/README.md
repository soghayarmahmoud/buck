# Font Files Placeholder

Download the following fonts from Google Fonts and place them in this directory:

## Required Fonts:

1. **Cairo** - https://fonts.google.com/specimen/Cairo
   - Cairo-Regular.ttf
   - Cairo-Bold.ttf

2. **Tajawal** - https://fonts.google.com/specimen/Tajawal
   - Tajawal-Regular.ttf
   - Tajawal-Bold.ttf

3. **Changa** - https://fonts.google.com/specimen/Changa
   - Changa-Regular.ttf
   - Changa-Bold.ttf

4. **Droid Arabic Naskh** - https://fonts.google.com/specimen/Droid+Arabic+Naskh
   - DroidArabicNaskh-Regular.ttf
   - DroidArabicNaskh-Bold.ttf

---

## Quick Download Steps:

1. Go to https://fonts.google.com
2. Search for the font name (e.g., "Cairo")
3. Click the font to open its page
4. Download all files (usually by clicking the download icon)
5. Extract the TTF files and place them in this directory

---

Note: After adding font files, run:
```
flutter clean
flutter pub get
flutter build apk --release
```
